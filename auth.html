Anyone can edit the IIDX SP ☆12(Lv12) rekidai data.

If you want to update rekidai data, please fork https://github.com/rekidai-info/rekidai-info.github.io, edit rekidai.json, and submit a Pull Request. That's all.
All you need to do is create a GitHub account, and you DO NOT need to install any git tools.
The rekidai.json file can be edited in your browser, and Pull Requests can be made in your browser.

Please submit at least one Pull Request to view this site.

誰でも IIDX SP ☆12(Lv12) 歴代データを修正できます。

歴代データを更新したい場合、https://github.com/rekidai-info/rekidai-info.github.io をフォークし、rekidai.json を編集し、Pull Request を提出してください。
必要なのはGitHub のアカウントを作成する事のみで、git などのツールをインストールする必要はありません。
rekidai.json ファイルはブラウザ上で編集が可能で、Pull Request もブラウザ上で提出できます。

本サイトを閲覧するには、一度以上 Pull Request を提出してください。
